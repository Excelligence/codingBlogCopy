textContent vs createTextNode

by Excelligence

Posted on Dec 15, 2018

Introduction

textContent can be used to change text of an existing element. This will replace existing content with new content. createTextNode can be used to add to existing text content.

HTML 


        <p id='para'>Lorem ipsum dolor sit amet laboriosam dicta fugit? </p>

JavaScript 


        const p =document.getElementById('para');

This will append. 


        p.append(document.createTextNode(' This created text node text'))

This will replace with new text 


        p.textContent='this is the changed text';
