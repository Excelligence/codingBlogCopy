CURL

 

Can be used for testing end points downloading files

 

for grabbing a url

 

curl <url>

 

to output to a file

 

curl -o <fileName> <url>

 

to include header information

 

curl -i <url>

 

to just get head

 

-I or --head

 

to download file with curl

 

curl -O <url>

 

Post request

is the post request url

 

curl -d "title=Hello&body=Hello World" <url>

 

PUT Request

curl -X PUT -d "title=Hello" <url>

 

Delete Request

curl -X DELETE <url>

 

For secure routes

 

curl -u <username>:<password> <url>

 

Follow redirection

curl -L <url>

 

FTP Protocol

 

Uploading a file using CURL

 

curl -u test@traversymedia.com:12345 -T ftp://ftp.traversymedia.com

 

curl -u : -T

Download files by ftp

 

curl -u : -O /
